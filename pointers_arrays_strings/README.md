# Pointers, Arrays, and Strings

## Description
This project focuses on understanding memory layouts, addresses, and variable scopes in the C programming language. It explores the practical implementation of pointers, character arrays, and advanced string manipulation techniques without utilizing the C standard library.

## Learning Objectives
By the end of this project, you will be able to explain:
* What pointers are and how to perform pointer arithmetic.
* The explicit differences between arrays and pointers in memory.
* How to manipulate and process strings using native indices.
* The boundaries and scope rules of low-level variables.

## Requirements
* **Environment**: Compiled on Ubuntu 20.04 LTS.
* **Compiler Options**: `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
* **Style Guidelines**: Code must adhere strictly to the Betty style standards.
