# Dynamic Memory Allocation: Malloc & Free

## Description
This project covers the concepts of automatic and dynamic memory allocation in C, specifically focusing on how to use `malloc` and `free` to manage memory on the heap without leaks.

## Learning Objectives
- Understand the difference between automatic and dynamic allocation.
- Know how to use `malloc` and `free` properly.
- Learn how to identify and prevent memory leaks using `valgrind`.

## Requirements
- Compiled on Ubuntu 20.04 LTS using `gcc` with standard flags.
- Codes comply with the Betty documentation and style guide.
- Global variables are forbidden.
